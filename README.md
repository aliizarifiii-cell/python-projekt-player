# Python-projekt – Spelarehantering -Player.py

# Beskrivning
Detta projekt är en del av en gruppprojekt i Python. 
Min del av projektet hanterar spelare/användare.

Man kan:
- Skapa nya användare
- Välja befintliga användare
- Spara användare i JSON-fil

# Funktioner i min kod

I min fil (`player.py`) visar jag:
- Variabler och datatyper
- If/elif/else villkor
- Loopar (for)
- Funktioner
- En klass (OOP)
- Filhantering med JSON
- Externt bibliotek: `rich`

## Hur man kör programmet

1. Installera bibliotek:
pip install rich

2. Kör programmet:
python player.py

# Min del av projektet

Jag ansvarade för filen:
- `player.py`

Den hanterar all logik för att skapa och välja spelare.
